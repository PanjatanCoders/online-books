Write a Java program to convert uppercase letters to lowercase.

Implementation Approaches

Java 
public class CaseConverter {
    
    // Method 1: Using built-in toLowerCase()
    public static String convertToLowercase(String input) {
        if (input == null) return null;
        return input.toLowerCase();
    }
    
    // Method 2: Manual conversion using ASCII
    public static String manualToLowercase(String input) {
        if (input == null || input.isEmpty()) return input;
        
        StringBuilder result = new StringBuilder();
        for (char c : input.toCharArray()) {
            if (c >= 'A' && c <= 'Z') {
                result.append((char) (c + 32));
            } else {
                result.append(c);
            }
        }
        return result.toString();
    }
    
    public static void main(String[] args) {
        String test = "HELLO World 123!";
        System.out.println("Original: " + test);
        System.out.println("Lowercase: " + convertToLowercase(test));
    }
}
Complexity: O(n) time where n is string length. Built-in method is optimized and recommended for production.

Find the second largest number in an array using Java.

Efficient Algorithm

Java 
public class SecondLargestFinder {
    
    // Single pass approach - Most Efficient
    public static Integer findSecondLargest(int[] array) {
        if (array == null || array.length < 2) {
            return null;
        }
        
        int largest = Integer.MIN_VALUE;
        int secondLargest = Integer.MIN_VALUE;
        
        for (int num : array) {
            if (num > largest) {
                secondLargest = largest;
                largest = num;
            } else if (num > secondLargest && num != largest) {
                secondLargest = num;
            }
        }
        
        return secondLargest == Integer.MIN_VALUE ? null : secondLargest;
    }
    
    public static void main(String[] args) {
        int[] array = {3, 1, 4, 1, 5, 9, 2, 6};
        System.out.println("Second Largest: " + findSecondLargest(array));
    }
}
Complexity:
  • Time: O(n) - Single pass through array
  • Space: O(1) - Constant extra space

Java (RestAssured) 
import io.restassured.RestAssured;
import static org.hamcrest.Matchers.*;

public class APIValidator {
    
    public void validateUserAPI() {
        RestAssured
            .given()
                .baseUri("https://api.example.com")
                .header("Content-Type", "application/json")
            .when()
                .get("/users/1")
            .then()
                .statusCode(200)
                .body("id", notNullValue())
                .body("name", notNullValue())
                .body("email", matchesPattern("^[A-Za-z0-9+_.-]+@(.+)$"))
                .body("age", greaterThan(0))
                .body("status", oneOf("active", "inactive"));
    }
}
Validation Checklist:
  • Status Code: Verify HTTP response codes
  • Schema: Match against JSON schema
  • Data Types: Ensure correct field types
  • Mandatory Fields: Check required fields
  • Response Time: Performance validation

Write a Java program to convert uppercase to lowercase and vice versa (Example: HexAwarE ‚Üí hEXaWARe).

Toggle Case Implementation

Java 
public class CaseToggler {
    
    public static String toggleCase(String input) {
        if (input == null || input.isEmpty()) {
            return input;
        }
        
        StringBuilder result = new StringBuilder();
        
        for (char c : input.toCharArray()) {
            if (Character.isUpperCase(c)) {
                result.append(Character.toLowerCase(c));
            } else if (Character.isLowerCase(c)) {
                result.append(Character.toUpperCase(c));
            } else {
                result.append(c); // Keep non-alphabetic
            }
        }
        
        return result.toString();
    }
    
    public static void main(String[] args) {
        String input = "HexAwarE";
        String output = toggleCase(input);
        
        System.out.println("Input:  " + input);
        System.out.println("Output: " + output);
        // Output: hEXaWARe
    }
}
Approach: Iterate through each character, check if uppercase/lowercase, and toggle accordingly. Non-alphabetic characters remain unchanged.

public void clickMenuItem(String menuText) {

    Actions actions = new Actions(driver);

Describe how to switch between frames in Selenium with a practical scenario.

Frame Switching Methods

Java 
public class FrameSwitcher {
    
    // Switch by index
    public void switchByIndex(int index) {
        driver.switchTo().frame(index);
    }
    
    // Switch by name or ID
    public void switchByNameOrId(String nameOrId) {
        driver.switchTo().frame(nameOrId);
    }
    
    // Switch by WebElement
    public void switchByElement(By locator) {
        WebElement frameElement = driver.findElement(locator);
        driver.switchTo().frame(frameElement);
    }
    
    // Switch back to main content
    public void switchToDefault() {
        driver.switchTo().defaultContent();
    }
    
    // Practical Scenario: Payment Gateway
    public void processPayment(String cardNumber, String cvv) {
        // Main page - click pay button
        driver.findElement(By.id("proceedToPay")).click();
        
        // Switch to payment iframe
        WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
        wait.until(ExpectedConditions.frameToBeAvailableAndSwitchToIt(
            By.xpath("//iframe[@id='payment-frame']")));
        
        // Fill payment details in iframe
        driver.findElement(By.id("card-number")).sendKeys(cardNumber);
        driver.findElement(By.id("cvv")).sendKeys(cvv);
        driver.findElement(By.id("submit-payment")).click();
        
        // Switch back to main content
        driver.switchTo().defaultContent();
        
        // Verify confirmation
        WebElement confirmation = wait.until(
            ExpectedConditions.presenceOfElementLocated(
                By.id("payment-confirmation")));
        System.out.println("Status: " + confirmation.getText());
    }
}
Switching Methods:
  • By Index: driver.switchTo().frame(0) - Not recommended
  • By Name/ID: driver.switchTo().frame("frameName")
  • By WebElement: Most reliable approach
  • Default Content: Returns to main page
Important: Always switch back to default content before switching to another frame

Explain Page Factory in Selenium and its annotations.

Page Factory Overview

Page Factory is a built-in Page Object Model implementation in Selenium that provides annotation-based element initialization.

Java 
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.support.FindBy;
import org.openqa.selenium.support.PageFactory;
import org.openqa.selenium.support.CacheLookup;
import org.openqa.selenium.support.How;

public class LoginPage {
    
    private WebDriver driver;
    
    // @FindBy - Basic annotation
    @FindBy(id = "username")
    private WebElement usernameField;
    
    @FindBy(name = "password")
    private WebElement passwordField;
    
    // Different locator strategies
    @FindBy(xpath = "//button[@type='submit']")
    private WebElement loginButton;
    
    @FindBy(css = ".error-message")
    private WebElement errorMessage;
    
    // Using How enum
    @FindBy(how = How.LINK_TEXT, using = "Forgot Password?")
    private WebElement forgotPasswordLink;
    
    // @CacheLookup - Cache element reference
    @FindBy(id = "logo")
    @CacheLookup
    private WebElement logo;
    
    // List of elements
    @FindBy(className = "menu-item")
    private List menuItems;
    
    // @FindBys - AND condition (all must match)
    @FindBys({
        @FindBy(className = "form-input"),
        @FindBy(name = "email")
    })
    private WebElement emailInput;
    
    // @FindAll - OR condition (any can match)
    @FindAll({
        @FindBy(id = "submit"),
        @FindBy(name = "submit"),
        @FindBy(xpath = "//button[@type='submit']")
    })
    private WebElement submitButton;
    
    // Constructor
    public LoginPage(WebDriver driver) {
        this.driver = driver;
        PageFactory.initElements(driver, this);
    }
    
    // Methods
    public void login(String username, String password) {
        usernameField.sendKeys(username);
        passwordField.sendKeys(password);
        loginButton.click();
    }
    
    public String getErrorMessage() {
        return errorMessage.getText();
    }
}

Key Annotations

Available Annotations:
  • @FindBy: Locates element using id, name, xpath, css, etc.
  • @FindBys: AND condition - all locators must match
  • @FindAll: OR condition - any locator can match
  • @CacheLookup: Caches element (use for static elements only)
Best Practice: Avoid @CacheLookup for dynamic elements. Elements are initialized lazily only when accessed.

What is the purpose of a constructor in Page Factory?

Constructor Purpose

The constructor in Page Factory serves multiple critical purposes for proper page object initialization.

Main Functions:
  • WebDriver Initialization: Receives and stores the driver instance
  • Element Initialization: Calls PageFactory.initElements() to initialize @FindBy elements
  • Lazy Initialization: Elements are located only when accessed (proxy pattern)
  • Setup Actions: Can include wait initialization, base URL setup

Constructor Examples

Java 
// Basic Constructor
public class LoginPage {
    private WebDriver driver;
    
    @FindBy(id = "username")
    private WebElement username;
    
    // Constructor
    public LoginPage(WebDriver driver) {
        this.driver = driver;
        // Initialize all @FindBy elements
        PageFactory.initElements(driver, this);
    }
}

// Advanced Constructor with Wait
public class BasePage {
    protected WebDriver driver;
    protected WebDriverWait wait;
    
    public BasePage(WebDriver driver) {
        this.driver = driver;
        this.wait = new WebDriverWait(driver, Duration.ofSeconds(10));
        PageFactory.initElements(driver, this);
    }
}

// Constructor with Custom Field Decorator
public class EnhancedPage {
    private WebDriver driver;
    
    public EnhancedPage(WebDriver driver) {
        this.driver = driver;
        // Custom timeout for element location
        PageFactory.initElements(
            new AjaxElementLocatorFactory(driver, 15), 
            this
        );
    }
}

// Usage in Tests
@Test
public void testLogin() {
    WebDriver driver = new ChromeDriver();
    driver.get("https://example.com");
    
    // Constructor is called here
    LoginPage loginPage = new LoginPage(driver);
    loginPage.login("user@test.com", "password");
}

How It Works

Concept 
Initialization Flow:

1. Test creates page object
   LoginPage page = new LoginPage(driver);

2. Constructor receives driver
   public LoginPage(WebDriver driver) {
       this.driver = driver;

3. PageFactory.initElements() called
       PageFactory.initElements(driver, this);
   }

4. Elements are wrapped in proxies
   - @FindBy elements become proxy objects
   - Actual element location happens on first access

5. Element usage triggers location
   page.usernameField.sendKeys("test");
   // Element located NOW, not during initialization
Key Concept: PageFactory.initElements() must be called in the constructor. Without it, @FindBy annotations won't work and you'll get NullPointerException.
Lazy Initialization Benefit: Elements are not located during page object creation. They're located only when accessed, improving performance and handling dynamic pages better.
